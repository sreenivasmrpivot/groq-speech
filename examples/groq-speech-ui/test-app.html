Features Implemented

  • Next.js 15 with TypeScript and Tailwind CSS
  • Real-time audio recording with MediaRecorder API
  • Audio processing and WAV conversion
  • Groq API integration (REST and WebSocket)
  • Mock API for demonstration without real API key
  • Single-shot and continuous recognition modes
  • Transcription and translation support
  • Multiple language support (10+ languages)
  • Performance metrics with timing breakdown
  • Interactive charts using Recharts
  • Responsive design for desktop and mobile
  • API key management and secure storage
  • Export functionality for results
  • Real-time performance monitoring
  • Error handling and user feedback

Performance Metrics Features

  • Microphone capture timing
  • API call duration tracking
  • Response processing time
  • Total end-to-end timing
  • Success/failure rate monitoring
  • Historical performance trends
  • Visual charts and graphs
  • Real-time metric updates

How to Use

  1. Open the application at http://localhost:3000
  2. Configure your API key or enable mock mode
  3. Select recognition mode (Single Shot or Continuous)
  4. Choose operation type (Transcription or Translation)
  5. Select your language
  6. Click "Start Recording" and speak into your microphone
  7. View results and performance metrics
  8. Export results as needed
